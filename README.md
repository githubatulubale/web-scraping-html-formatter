<h2>ğŸ“Œ Project Overview</h2>
<p>
<b>WebSleuth</b> is a simple yet powerful <b>web scraping project</b> built using
<b>Python, Requests, and BeautifulSoup</b>.
The project fetches HTML content from any user-provided URL,
formats it into a readable structure, and saves it locally as an HTML file.
</p>

<p>
This project demonstrates the core concepts of
<b>HTTP requests, HTML parsing, and content formatting</b>,
which are essential skills for data collection and web automation.
</p>

<h2>ğŸ§  Problem Statement</h2>
<p>
Raw HTML pages fetched from the web are often unstructured and difficult to read.
Manually inspecting or cleaning such data is inefficient.
</p>

<p>
This project solves the problem by:
</p>
<ul>
  <li>Fetching live webpage content</li>
  <li>Parsing HTML using BeautifulSoup</li>
  <li>Beautifying and structuring HTML</li>
  <li>Saving formatted output to a file</li>
